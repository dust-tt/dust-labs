üìã Installation Instructions

For Excel Add-in:

  1. Download the Excel manifest file using the button above
  2. Open Excel and go to Insert ‚Üí Get Add-ins (or Insert ‚Üí My Add-ins)
  3. Click on Manage My Add-ins ‚Üí Upload My Add-in
  4. Browse and select the downloaded excel-manifest.xml file
  5. Click Upload to install the add-in
  6. The add-in will appear in the Home tab ribbon

For PowerPoint Add-in:

  1. Download the PowerPoint manifest file using the button above
  2. Open PowerPoint and go to Insert ‚Üí Get Add-ins (or Insert ‚Üí My Add-ins)
  3. Click on Manage My Add-ins ‚Üí Upload My Add-in
  4. Browse and select the downloaded powerpoint-manifest.xml file
  5. Click Upload to install the add-in
  6. The add-in will appear in the Home tab ribbon
Note: These add-ins require Office 365 or Office 2016 or later. For development and testing, ensure your local server is running before using the add-ins.

Alternative: Sideload for Development

  1. For development purposes, you can also sideload the add-in directly
  2. Place the manifest file in a network share or local folder
  3. Add the folder path to the Trusted Add-in Catalogs in Office
  4. Go to File ‚Üí Options ‚Üí Trust Center ‚Üí Trust Center Settings ‚Üí Trusted Add-in Catalogs
  5. Add the folder URL/path and check "Show in Menu"
