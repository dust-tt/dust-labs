Installation Instructions

For Excel Add-in:

  1. Download the Excel manifest file using the button above
  2. Open Excel and go to Insert ‚Üí Add-ins ‚Üí Other Add-ins ‚Üí My Add-ins
  3. Click on Upload My Add-in
  4. Browse and select the downloaded manifest.xml file
  5. The add-in will be installed
  6. The add-in will appear in the Home tab ribbon

For PowerPoint Add-in:

  1. Download the PowerPoint manifest file using the button above
  2. Open PowerPoint and go to Insert ‚Üí Add-ins ‚Üí Advanced ‚Üí Upload
  3. Browse and select the downloaded manifest.xml file
  4. The add-in will be installed
  5. The add-in will appear in the Home tab ribbon

For Power Automate Connectors:

  1. Download the Power Automate connectors zip file using the button above
  2. Go to the Power Automate portal and go to Solutions ‚Üí Import the solution
  3. Browse and select the downloaded PowerAutomate_DustAssistantSolution.zip file
